#### 简介
本项目旨在提供一个使能机密计算解决方案所需软件（包括固件、操作系统、虚拟化管理、机密虚机以及远程证明等）和仿真硬件的全局视角，提供运行或集成上述组件的完整参考模型，帮助用户集成使能机密计算解决方案，提前布局ARM CCA软件生态。
